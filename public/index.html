API

/inbox/

An Event Notifications LDN Inbox accepting Event Notification Offers.

Example

curl -X POST -H 'Content-Type: application/ld+json' --data-binary '@examples/announce.jsonld' http://localhost:8000/inbox/
where offer.jsonld: 
{ 
    "@context": "https://www.w3.org/ns/activitystreams" , 
    "id": "urn:uuid:6E5FAF88-A7F1-47A4-B087-77345EBFF495",
    "type": "Announce",
    "actor": {
        "id": "https://acme.org/profile/card#us",
        "inbox": "https://acme.org/inbox/",
        "type": "Organization"
    },
    "object": {
        "id": "157.193.101.1"
    }
}
